# Joule Development - Team Setup Guide

> Internal team setup steps for Joule development environment.

---

# 1. Install Python

## Requirement
Python 3.8+

## Download
https://www.python.org/downloads/

---

# 2. Install Node.js

## Download
https://nodejs.org/en/download

## Verify Installation

```bash
node -v
npm -v
```

---

# 3. Install Joule CLI

## Official Documentation
https://help.sap.com/docs/joule/joule-development-guide-ba88d1ec6a1b442098863d577c19b0c0/install-and-update-joule-studio-cli?locale=en-US&q=joule+cli

---

## Install Latest Version

```bash
npm install -g @sap/joule-studio-cli
```

---

## Verify Installation

```bash
joule -V
```

---

# 4. Setup SAP BTP Account

## Environment

```text
Home → Private Cloud Canary → SAP BTP Cockpit
```

---

# 5. Verify Destination Access

Check access to SAP system destination configured in BTP.

Example:

```text
Destination_ERX_001
```
