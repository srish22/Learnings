# 2. Admin / Tenant Set up

> Usually handled by administrator/platform team.


## Required Setup

- Joule service subscription instance created
- SAP product supporting Joule integration available
- Identity Authentication integrated
- SAP BTP global account available
- Same SAP Cloud Identity Services tenant used
- Required entitlements configured

> For more info: [Joule Onboarding Documentation](https://help.sap.com/docs/joule/integrating-joule-with-sap/onboarding-joule)
---

## Important Services

- SAP BTP
- Cloud Identity Services
- Identity Authentication
- Identity Provisioning
- SAP Build Work Zone

---

EU10 or EU30 subaccounts required

# 4. Development Approaches

There are two main ways to develop/manage Joule assistants.

<br><br>

---

## Option 1 — Joule Studio CLI

| What It Is | Supports | Best For | Main Advantages |
| :--- | :--- | :--- | :--- |
| - Terminal-based CLI tool for Joule development | - Local development<br><br>- Compilation<br><br>- Deployment<br><br>- CI/CD integration<br><br>- Secure authentication | - Advanced developers<br><br>- Automation<br><br>- CI/CD pipelines<br><br>- Git-based workflows | - Lightweight<br><br>- Terminal-driven<br><br>- Easy automation<br><br>- Enterprise pipeline friendly |


<br>

### CLI Setup

| Action | Command / Link |
| :--- | :--- |
| Install Python 3.8+ | https://www.python.org/downloads/ |
| Install Node.js | https://nodejs.org/en/download |
| Verify Node.js | `node -v`<br><br>`npm -v` |
| Install Latest Joule CLI | `npm install -g @sap/joule-studio-cli` |
| Install Specific CLI Version | `npm install -g @sap/joule-studio-cli@<version>` |
| Verify Joule CLI | `joule -V` |


<br>

---

## Option 2 — Joule Studio Code Editor

| What It Is | Supports | Best For | Main Advantages |
| :--- | :--- | :--- | :--- |
| - VS Code extension/tooling for Joule | - Templates<br><br>- Validation<br><br>- Quick Fix<br><br>- Schema updates<br><br>- Linting<br><br>- Deployment<br><br>- Assistant management | - Visual development<br><br>- Beginners<br><br>- Guided onboarding<br><br>- Faster learning | - Easier onboarding<br><br>- Visual workflows<br><br>- Integrated tooling<br><br>- Beginner friendly |


<br>

### Editor set up

| Action | Command / Link |
| :--- | :--- |
| Install VS Code | https://code.visualstudio.com/download |
| Install Node.js | https://nodejs.org/en/download |
| Verify Node.js | `node -v`<br><br>`npm -v` |
| Install Joule Extension | Search `Joule` in VS Code Extensions Marketplace |
| Open Extensions Panel | `Ctrl + Shift + X` |
| Login To Joule | Use Joule extension login flow |


---

# Which Approach Should I Use?

| Scenario | Recommended |
| :--- | :--- |
| Quick onboarding | Code Editor |
| Visual development | Code Editor |
| Learning Joule basics | Code Editor first |
| CI/CD automation | CLI |
| Git-heavy workflow | CLI |
| Enterprise deployments | CLI |
