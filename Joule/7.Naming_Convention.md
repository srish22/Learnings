## Capability DTA

| Resource Type | Character Length | Constraints |
| :--- | :--- | :--- |
| Capability name | 50 | It can contain only alphanumeric and underscore characters.<br><br>Spaces and consecutive underscores are not allowed. |
| Capability namespace | 100 | It can contain only alphanumeric characters. |
| Capability version | 20 | Semantic version qualifier → only `SNAPSHOT` supported |
| Capability display name | 128 | Can contain alphanumeric, underscore and hyphen characters.<br><br>Cannot contain carriage return (`\r`), newline (`\n`) or tab (`\t`). |
| Capability description | 512 | Cannot contain carriage return (`\r`), newline (`\n`) or tab (`\t`). |
| Capability context variable | 64 | Can contain alphanumeric, underscore and hyphen characters.<br><br>Cannot contain carriage return (`\r`), newline (`\n`) or tab (`\t`). |
| Resource file names | 30 | Can contain only alphanumeric and underscore characters.<br><br>Spaces and consecutive underscores are not allowed. |
| Resource grouping folder names | 30 | Can contain only alphanumeric and underscore characters.<br><br>Spaces and consecutive underscores are not allowed. |
| Resource descriptions | 128 | Cannot contain carriage return (`\r`), newline (`\n`) or tab (`\t`).<br><br>Formatting using block scalars is allowed. |
| Context key (dialog functions) | 64 | Must contain only alphanumeric and underscore characters.<br><br>Must not contain:<br><br>- `public`<br>- `private`<br><br>Must not be equal to:<br><br>- `request`<br>- `skip_user_input`<br>- `transient`<br>- `user`<br>- `conversation` |
| Dialog function properties (system alias, parameter name, result-variable) | 64 | Can contain only alphanumeric and underscore characters. |
| Dialog function API request path | 2048 | Should be encoded. |
