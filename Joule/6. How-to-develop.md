# 1. High-Level Flow

```text
Assistant
    ↓
Capabilities
    ↓
Scenarios
    ↓
Dialog Functions
```

<br>

---

## Core Concepts

| Concept | Meaning |
| :--- | :--- |
| Assistant | Collection of capabilities assembled together |
| Capability | Group of related business skills + language understanding |
| Scenario | Semantic representation of a specific use case |
| Dialog Function | Executable block that performs tasks and returns data/UI |
| DTA (Design Time Artifact) | YAML-based source definition of capabilities/resources |

<br>

---

### DTA


| What Is A DTA? | What Does It Contain? | Why Is It Important? |
| :--- | :--- | :--- |
| - Full form → Design Time Artifact<br><br>- YAML-based source artifact<br><br>- Human-readable capability definition | - Dataset<br><br>- Dialog flow<br><br>- Configurations<br><br>- Assistant configuration YAML | - Treated like source code<br><br>- Git/version control friendly<br><br>- Supports collaboration<br><br>- Enables merge/conflict handling<br><br>- Provides audit trail |

# 5. Capability

| Topic | Details |
| :--- | :--- |
| Meaning | Group of related business skills |
| Includes | Skills + language understanding |
| Example | Purchasing capability → order creation + delivery status |
| Development Style | YAML-based source development |
| Source Management | Git/version control friendly |

<br><br>

---

# 6. Scenario

| Topic | Details |
| :--- | :--- |
| Meaning | Semantic representation of a use case |
| Contains | Use case description<br><br>Semantic relations<br><br>Execution target |
| Target | Usually mapped to function execution |

<br><br>

---

# 7. Dialog Function

| Topic | Details |
| :--- | :--- |
| Meaning | Executable instruction block |
| Purpose | Performs multiple tasks |
| Output | Returns data or UI model |
| Displayed To | End user |

| Key Takeaways |
| :--- |
| - Joule development is YAML-driven<br><br>- DTA acts like source code<br><br>- Assistants are built using capabilities<br><br>- Capabilities contain scenarios and dialog functions<br><br>- Git collaboration/versioning is an important development model |
