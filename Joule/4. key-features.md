# SAP Joule — Key Features

> **Source**: [SAP Joule Feature Summary Document (Official PDF)](https://help.sap.com/doc/0a1c2b2b16f843dca835b44a5db033be/CLOUD/en-US/Joule_FSD.pdf)

## Key Features

- **Integration with SAP applications** — Natively embedded across SAP cloud products (SuccessFactors, S/4HANA Cloud, BTP, Ariba, etc.)
- **Conversational UI** — Chat-based interface; interact using natural language, no T-codes or menu navigation needed
- **Conversation management** — Manage and organize your conversations with Joule within the interface
- **Uploading of files** — Attach and share files directly within the Joule conversation
- **Conversation history** — Access and revisit past conversations with Joule
- **Conversational search** — Search across SAP data and content using natural language queries
- **Personalized experience** — Joule adapts to your role, context, and SAP authorization profile
- **Access on mobile devices** — Available on mobile via SAP Mobile Start (also supports Siri on iOS)
- **Feedback** — Provide feedback on Joule responses to improve accuracy
- **Document grounding** — Answers are grounded in real business documents (uses RAG to prevent hallucinations)
- **Integration with third-party generative AI assistants** — Works with external AI tools (e.g. Microsoft 365 Copilot — bidirectional integration)
- **Conversational insights (analytics)** — Ask business questions and get data-driven insights in natural language
- **Administrative settings** — Admins can configure and control Joule behavior, access, and scope
- **Data management** — Manage data sources and permissions that Joule can access
- **Pro-code development tools** — Developers can use Joule to generate code, build SAP apps, and extend functionality
